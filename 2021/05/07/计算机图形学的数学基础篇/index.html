计算机图形学中的直线

首先要明确的是，这里要叙述的直线都是在二维空间的情况下。

假设平面中有一直线 l ，其中P，Q是该直线中的两点。

直线的参数式表示

设参数为t，直线上任一点X可表示为：
$$
X = (1-t)P + tQ
$$
该式也正是线性插值的公式。

这种表示方法在三维空间里也同样有效，其证明也十分简单，将式子变形为：
$$
X = P + t * (Q - P)
$$
即P点沿着PQ的方向行进了t单位。

该表示式中，当 t < 0 说明X在P点前；当 t > 1说明X在Q点后，当 0 <= t <= 1说明X点在以PQ为两端点的线段上。
