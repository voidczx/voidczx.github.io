这里的origin是我们之前设置的远端仓库名，后面的master是分支名，当我们用git init命令初始化本地仓库后，每一次键入命令的时候都会在提示串右边看到一个括号，括号里的名字即是我们的分支名。

这个命令类似于本地中的commit，但并非一次上交一个版本，而是把本地仓库所以的版本一起提交到远端仓库。（这点未经仔细验证，大致如此，但仍存疑）
